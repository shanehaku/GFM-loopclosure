# GFM Whole Map Viewer

GitHub Pages static viewer for whole-map comparison only.

Open `index.html`, then enter `maps/whole_map/`.

Groups:

- Original: `original`, `no loop closure`
- Optimized: `Sim3 Graph`, `2node submap`, `structure deformation`

The viewer lazy-loads only the selected map group.

See `ASSET_PLACEMENT.md` for PLY placement.
